Dossier des données

Choisissez où stocker les bases de données, fichiers et configuration. Vous pouvez créer un nouveau dossier.

Prérequis

Vérifiez que les logiciels nécessaires sont installés. Cliquez sur les liens pour télécharger ce qui manque.

    Configuration de base

    Remplissez les paramètres. Un fichier .env complet sera créé dans le dossier des données.

    PostgreSQL (conteneur local)
    JWT (sécurité)
    ECR (mises à jour depuis AWS)
    AWS (credentials pour ECR et S3)
    S3 (stockage cloud, optionnel)

    Télécharger les images

    Téléchargez dès maintenant la dernière version des conteneurs depuis ECR (recommandé si vous avez configuré ECR).

    Installation terminée

    Tout est configuré. Vous pouvez lancer Parallele SchoolMatrix ou quitter.
